AUST WiFi 自动重连工具 - 安装向导

欢迎使用 AUST WiFi 自动重连工具！
===============================================

🎓 专为安徽理工大学校园网设计
📡 智能检测网络状态，自动重连
🚀 一次配置，终身使用
💻 后台静默运行，开机自启动

安装前准备：
-----------
✓ 确认您的操作系统：Windows 10/11 (64位)
✓ 确认您是安徽理工大学在校学生
✓ 准备您的校园网账号信息（学号和密码）
✓ 确保您有管理员权限（用于设置开机自启）

安装内容：
---------
本安装程序将为您安装以下组件：
• AUST_WIFI.exe - 主程序文件
• 系统托盘图标和相关资源
• Qt运行库（如需要）
• 程序配置和设置功能

安装位置：
---------
默认安装路径：C:\Program Files\AUST_WIFI\
您也可以选择自定义安装路径

磁盘空间要求：
------------
• 最小空间：50 MB
• 推荐空间：100 MB

网络运营商支持：
--------------
• 中国电信 (aust)
• 中国联通 (unicom)  
• 中国移动 (cmcc)

重要提醒：
---------
⚠️ 请确保在校园网环境下使用本软件
⚠️ 请遵守学校网络使用规定
⚠️ 首次运行需要配置您的账号信息
⚠️ 建议关闭杀毒软件的实时保护以避免误报

技术支持：
---------
如遇到问题，请参考安装后的使用说明或README文档。
