========================================================
      AUST WiFi 自动重连工具 - 安装向导 v1.0
========================================================

🎓 欢迎使用 AUST WiFi 自动重连工具！

专为安徽理工大学校园网设计的智能网络管理软件
基于 Qt6/C++ 开发，提供自动检测、智能重连功能

========================================================
                   🖥️ 系统要求
========================================================

【硬件要求】
✓ 处理器：Intel/AMD 64位处理器
✓ 内存：最低 2GB RAM，推荐 4GB+
✓ 存储空间：100MB 可用磁盘空间
✓ 网络：WiFi或有线网络支持

【操作系统】
✓ Windows 10 Build 1903 或更高版本
✓ Windows 11 (所有版本)
✓ 64位系统架构

【软件环境】
✓ Microsoft Visual C++ 2019+ Redistributable
✓ .NET Framework 4.7.2+ (通常已预装)
✓ Windows Defender 或其他杀毒软件例外设置

========================================================
                   🌐 网络环境要求
========================================================

【适用网络】
✓ 安徽理工大学校园网（宿舍网络）
✓ 图书馆WiFi网络
✓ 实验室有线网络
✓ 教学楼公共WiFi

【运营商支持】
✓ 中国电信校园套餐 → 选择"电信(aust)"
✓ 中国联通校园套餐 → 选择"联通(unicom)"
✓ 中国移动校园套餐 → 选择"移动(cmcc)"

【账号要求】
✓ 有效的学号（完整格式，如：2023305123）
✓ 校园网登录密码
✓ 确保账户未欠费或被停用

========================================================
                   📦 安装内容
========================================================

本安装程序将在您的计算机上安装：

【核心组件】
• AUST_WIFI.exe - 主程序文件 (约5MB)
• app.ico - 应用程序图标
• Qt运行库 - 界面框架支持 (约20MB)

【系统集成】
• 桌面快捷方式
• 开始菜单程序组
• 系统托盘功能
• 开机自启动设置（可选）

【配置文件】
• Windows注册表配置项
• 用户凭据加密存储
• 程序设置和偏好

========================================================
                   📂 安装路径选择
========================================================

【默认安装路径】
C:\Program Files\AUST_WIFI\

【自定义路径要求】
✓ 避免包含中文字符的路径
✓ 确保有写入权限
✓ 推荐安装到程序文件目录

【磁盘空间】
• 最小空间：50 MB
• 推荐空间：100 MB（含运行库）
• 临时文件：20 MB（安装过程中）

========================================================
                   ⚠️ 安装前准备
========================================================

【必需操作】
1. 关闭所有杀毒软件的实时保护
2. 确保以管理员身份运行安装程序
3. 关闭其他网络管理软件
4. 准备您的校园网账号信息

【可选操作】
1. 备份现有网络配置
2. 记录当前网络连接设置
3. 关闭不必要的后台程序

【网络测试】
建议安装前先手动测试校园网登录：
1. 连接到校园网
2. 打开浏览器访问任意网站
3. 在认证页面输入学号密码
4. 确认能够正常上网

========================================================
                   🔒 安全和隐私
========================================================

【数据安全】
✓ 密码使用Windows DPAPI加密存储
✓ 所有配置信息仅保存在本地
✓ 不向任何第三方服务器发送数据
✓ 支持完全卸载和数据清理

【网络安全】
✓ 仅连接安徽理工大学官方认证服务器
✓ 支持HTTPS加密传输协议
✓ 不监听任何网络端口
✓ 不修改系统网络配置

【隐私保护】
✓ 不收集用户个人信息
✓ 不上传使用统计数据
✓ 不包含任何广告或跟踪代码
✓ 开源代码可供审查

========================================================
                   🚀 安装后设置
========================================================

首次运行程序时，请按以下步骤配置：

【步骤1：基本信息】
• 学号：输入完整学号（如：2023305123）
• 密码：输入校园网登录密码
• 运营商：根据您的套餐选择对应运营商

【步骤2：高级设置】
• ☑ 开机自启动（推荐勾选）
• ☑ 后台运行（推荐勾选）
• ☑ 自动重连（推荐勾选）

【步骤3：测试连接】
• 点击"保存"按钮保存配置
• 程序将自动测试网络连接
• 确认托盘图标显示正常状态

========================================================
                   🛠️ 故障排除
========================================================

【常见安装问题】

问题：安装程序无法启动
解决：右键选择"以管理员身份运行"

问题：提示缺少运行库
解决：安装Microsoft Visual C++ Redistributable

问题：杀毒软件误报
解决：将程序添加到白名单或暂时关闭实时保护

问题：安装路径权限不足
解决：选择其他路径或以管理员身份安装
