========================================================
      AUST WiFi 自动重连工具 - 安装向导 v2.0
========================================================

🎓 欢迎使用 AUST WiFi 自动重连工具 v2.0！

专为安徽理工大学校园网设计的智能网络管理软件
基于 Qt6/C++ 开发，全新v2.0重大更新，现已支持学生和教师双用户类型

🌟 v2.0 核心特性：
✅ 双配置智能管理 - 同时保存学生和教师配置，自动切换
✅ WiFi智能检测 - 根据网络SSID自动识别用户类型
✅ 企业级稳定性 - 智能重试机制，3次重试+30秒冷却期
✅ 专业用户体验 - 全新Qt Designer界面，完美布局
✅ 零感知体验 - AUST_Student → 学生登录，AUST_Faculty → 教师登录

========================================================
                   🖥️ 系统要求
========================================================

【硬件要求】
✓ 处理器：Intel/AMD 64位处理器
✓ 内存：最低 2GB RAM，推荐 4GB+
✓ 存储空间：100MB 可用磁盘空间
✓ 网络：WiFi或有线网络支持

【操作系统】
✓ Windows 10 Build 1903 或更高版本
✓ Windows 11 (所有版本)
✓ 64位系统架构 (x64 compatible)

【软件环境】
✓ Microsoft Visual C++ 2019+ Redistributable
✓ .NET Framework 4.7.2+ (通常已预装)
✓ Windows Defender 或其他杀毒软件例外设置

========================================================
                   🌐 网络环境要求
========================================================

【适用网络】
✓ 安徽理工大学校园网（宿舍网络）
✓ 图书馆WiFi网络（AUST_Student/AUST_Faculty）
✓ 实验室有线网络
✓ 教学楼公共WiFi

【智能网络检测v2.0】
🔥 新特性：WiFi智能检测
• AUST_Student WiFi → 自动使用学生配置
• AUST_Faculty WiFi → 自动使用教师配置
• 其他网络 → 根据用户首选项选择配置

【网络支持】
✓ 学生用户：
  - 中国电信校园套餐 → 选择"电信(aust)"
  - 中国联通校园套餐 → 选择"联通(unicom)"
  - 中国移动校园套餐 → 选择"移动(cmcc)"
✓ 教师用户：
  - 自动使用学校统一网络服务器(jzg)
  - 无需选择运营商

【账号要求】
✓ 学生：有效的学号（完整格式，如：2023305123）
✓ 教师：有效的工号（如：2021043）
✓ 校园网登录密码
✓ 确保账户未欠费或被停用

                   📂 安装路径选择

【自定义路径要求】

✓ 确保有写入权限

【磁盘空间】
• 最小空间：60 MB

• 临时文件：25 MB（安装过程中）

========================================================
                   ⚠️ 安装前准备
========================================================

【必需操作】
1. 关闭所有杀毒软件的实时保护
2. 确保以管理员身份运行安装程序
3. 关闭其他网络管理软件
4. 准备您的校园网账号信息（学生和教师可同时配置）

【可选操作】
1. 备份现有网络配置
2. 记录当前网络连接设置
3. 关闭不必要的后台程序

【网络测试】
建议安装前先手动测试校园网登录：
1. 连接到校园网（AUST_Student或AUST_Faculty）
2. 打开浏览器访问任意网站
3. 在认证页面输入学号/工号和密码
4. 确认能够正常上网

========================================================
                   🛠️ 故障排除
========================================================

【常见安装问题】

问题：安装程序无法启动
解决：右键选择"以管理员身份运行"

问题：提示缺少运行库
解决：安装Microsoft Visual C++ Redistributable

问题：杀毒软件误报
解决：将程序添加到白名单或暂时关闭实时保护

问题：安装路径权限不足
解决：选择其他路径或以管理员身份安装

问题：双配置设置失败
解决：检查Windows用户权限，以管理员身份重新配置
