# AUST WiFi Android v1.0.6

## 更新内容

- 修复自动登录过早触发时，认证网关返回 HTTP 200 但内容未确认成功，导致显示“服务器响应异常”的问题。
- 自动登录现在会在识别到校园网环境后等待网络稳定，再发起登录；遇到 HTTP 200 非成功响应时会短延迟自动重试。
- 更新检查会避开登录和 WiFi 网络绑定阶段，减少校园网重定向造成的 SSL 证书错误提示。
- 放宽登录成功响应识别，兼容字符串形式的 `result` 和中文成功提示。

## 发布信息

- Version Name: `1.0.6`
- Version Code: `16`
- 平台: Android arm64-v8a
