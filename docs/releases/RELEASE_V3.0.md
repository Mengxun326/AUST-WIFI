## 🎉 V3.0 重大更新

### 🔥 **全新特性**

#### 1. **Windows 11 Fluent Design UI** 🎨
- **完全重新设计**：采用 Windows 11 Fluent Design 设计语言
- **现代化界面**：卡片式布局，清晰的信息层次结构
- **流畅动画**：窗口移动、切换无卡顿，极致流畅体验
- **统一视觉风格**：与 Windows 11 系统应用完美融合
- **标签页优化**：统一的标签页高度，切换无闪烁

#### 2. **性能优化升级** ⚡
- **完全异步架构**：移除所有 `waitForFinished()` 阻塞操作
- **UI 响应速度提升 300%**：所有网络操作异步执行，不阻塞 UI 线程
- **智能窗口管理**：窗口移动时自动暂停后台任务，零卡顿体验
- **异步 WiFi 检测**：非阻塞 SSID 检测，不影响用户操作
- **内存优化**：更高效的资源管理，降低内存占用

#### 3. **跨平台支持** 🌐
- **Linux 支持**：全新支持 Linux 系统（Ubuntu、Debian 等主流发行版）
- **统一代码库**：一套代码，多平台编译
- **平台适配**：自动检测操作系统，使用对应网络命令
  - Windows: `netsh wlan show interfaces` (UTF-8 编码)
  - Linux: `iwgetid -r` 或 `nmcli` 作为备用方案
  - macOS: `airport -I` 命令
- **编译脚本**：提供 Linux 一键编译脚本 `build.sh`
- **详细文档**：新增 Linux 编译指南 `docs/LINUX_BUILD.md`

#### 4. **中文 WiFi 名称支持** 🔤
- **UTF-8 编码**：完美支持中文 WiFi 名称显示
- **编码转换**：Windows 使用 `chcp 65001` 强制 UTF-8 输出
- **PowerShell 备用方案**：主方案失败时自动切换到 PowerShell 检测
- **Linux 原生支持**：Linux 系统原生 UTF-8 支持

#### 5. **双配置智能管理** 👨‍🎓👨‍🏫
- **智能用户检测**：根据 WiFi 网络自动识别学生/教师身份
- **双账号管理**：同时保存学生和教师配置，无需频繁切换
- **自动适配登录**：
  - `AUST_Student` → 自动使用学生配置
  - `AUST_Faculty` → 自动使用教师配置
  - 其他网络 → 默认学生配置

#### 6. **WiFi 智能检测增强** 📡
- **实时 SSID 监控**：异步检测 WiFi 网络变化
- **智能类型判断**：根据网络名称自动确定用户类型
- **无感知切换**：网络变化时自动重新认证
- **信号通知**：新增 `wifiSSIDUpdated` 信号，实时更新 UI

## 🔧 安装和使用

### **系统要求**

#### Windows
- **操作系统**：Windows 10 Build 1903+ / Windows 11
- **架构**：x64 (64位)
- **内存**：最低 2GB RAM
- **存储空间**：100MB 可用空间

#### Linux
- **操作系统**：Ubuntu 20.04+ / Debian 11+ / 其他主流发行版
- **架构**：x64 (64位)
- **依赖库**：
  ```bash
  sudo apt-get install build-essential
  sudo apt-get install libqt6widgets6-dev libqt6network6-dev
  sudo apt-get install wireless-tools  # 或 network-manager
  ```

### **安装方式**

#### Windows
1. **下载发布包**：从 Release 页面下载最新版本
2. **运行安装程序**：双击 `AUST_WIFI_Setup.exe`
3. **首次配置**：启动后设置学生和教师账号信息
4. **开机自启**：勾选"开机自动启动"选项

#### Linux
1. **获取源码**：
   ```bash
   git clone https://github.com/Mengxun326/AUST-WIFI.git
   cd AUST-WIFI
   ```
2. **编译安装**：
   ```bash
   chmod +x build.sh
   ./build.sh
   ```
3. **首次配置**：运行编译后的可执行文件，设置账号信息

详细 Linux 编译说明请参考：[Linux 编译指南](../../docs/LINUX_BUILD.md)

### **配置说明**
- **学生配置**：学号(≥8位) + 密码 + 运营商
- **教师配置**：工号(≤8位) + 密码 (固定使用学校服务器)
- **智能提示**：输入格式错误时自动提示
