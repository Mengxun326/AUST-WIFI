# AUST WiFi Android v1.0.2

## 更新内容

- 支持通过校园认证网关识别校园网环境。用户使用路由器接入校园网、WiFi 名称变为自定义名称时，只要 `10.255.0.19` 认证网关可达，仍可自动登录。
- Android 界面文案从“校园 WiFi”调整为“校园网环境/校园网络”，避免把路由器接入场景误判为不支持。
- 保留原有 AUST SSID 识别逻辑；官方 `AUST_Student`、`AUST_Faculty` 仍会立即识别。

## 发布信息

- Version Name: `1.0.2`
- Version Code: `12`
- 平台: Android arm64-v8a
