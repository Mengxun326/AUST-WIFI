【首次配置向导】
程序首次启动时会自动显示配置界面，请按照以下步骤设置：

┌─────────────────────────────────────┐
│      🔧 配置步骤详解                │
└─────────────────────────────────────┘

【第1步：基本信息设置】
• 学号输入框：
  - 输入您的完整学号（例如：2023305123）
  - 必须包含所有数字，不能省略前缀0
  - 确保与校园网认证系统中的学号一致

• 密码输入框：
  - 输入您的校园网登录密码
  - 注意：这是校园网密码，不是学号密码
  - 密码将被安全加密存储，仅在本地使用

• 运营商选择：
  ┌─────────────────────────────────┐
  │ 电信用户 → 选择"电信(aust)"     │
  │ 联通用户 → 选择"联通(unicom)"   │
  │ 移动用户 → 选择"移动(cmcc)"     │
  └─────────────────────────────────┘

【第2步：高级选项设置】
• ☑ 开机自启动
  - 建议勾选：系统启动时自动运行程序
  - 需要管理员权限，首次设置可能需要确认

• ☑ 后台运行
  - 建议勾选：程序最小化到系统托盘
  - 不占用桌面空间，在后台静默工作

【第3步：保存并测试】
• 点击"保存"按钮完成配置
• 程序将自动开始网络监控
• 查看系统托盘图标确认运行状态

========================================================
                   💻 日常使用指南
========================================================

【系统托盘功能】
程序运行后会在系统托盘（屏幕右下角）显示图标

🎨 托盘图标状态指示：
• 🟢 绿色：网络连接正常，一切工作正常
• 🔴 红色：网络断开，程序正在尝试重连
• 🟡 黄色：正在检测网络状态或连接中
• ⚫ 灰色：程序已停止监控功能

【右键菜单操作】
右击系统托盘图标，可以进行以下操作：

┌─────────────────────────────────────┐
│ 📝 显示配置                          │
│ → 重新打开配置界面修改设置           │
├─────────────────────────────────────┤
│ ▶️ 开始自动重连                     │
│ → 启动网络监控和自动重连功能         │
├─────────────────────────────────────┤
│ ⏸️ 停止自动重连                     │
│ → 暂停网络监控，但程序继续运行       │
├─────────────────────────────────────┤
│ ❌ 退出程序                         │
│ → 完全关闭程序，停止所有功能         │
└─────────────────────────────────────┘

【自动重连原理】
• 检测频率：每1秒检测一次网络状态
• 检测方法：向可靠的网站发送请求测试连通性
• 重连触发：检测到网络断开时立即发送登录请求
• 智能重试：避免重复请求，确保高效稳定

========================================================
                   🔧 故障排除指南
========================================================

【常见问题及解决方案】

❓ 问题1：程序启动后没有托盘图标
💡 解决：
  • 检查Windows系统托盘设置
  • 确保系统托盘功能已启用
  • 重新启动程序

❓ 问题2：配置界面无法显示
💡 解决：
  • 以管理员身份运行程序
  • 删除注册表配置项重新配置
  • 检查防火墙是否阻止程序运行

❓ 问题3：网络检测失败
💡 解决：
  • 确认已连接到校园网
  • 检查学号密码是否正确
  • 确认运营商选择是否正确
  • 手动测试浏览器是否能正常上网

❓ 问题4：自动重连不工作
💡 解决：
  • 检查程序是否已启动自动重连功能
  • 确认托盘图标显示正常状态
  • 重新保存配置信息
  • 检查网络环境是否支持

❓ 问题5：开机自启动失败
💡 解决：
  • 以管理员身份重新设置
  • 检查Windows启动项设置
  • 手动添加到启动文件夹
